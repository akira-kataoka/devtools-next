# 📜 行動規範 (Code of Conduct)

このプロジェクトは Contributor Covenant v2.1 を基準としつつ、Salesforce 業務担当者・管理者・開発者の協働を前提に独自運用ルールを加えています。

---

## 🤝 私たちの約束

DevToolsNext コミュニティの参加者・貢献者・メンテナとして、私たちは年齢・体型・障害の有無・民族性・性自認・性表現・経験レベル・国籍・容姿・人種・宗教・性的指向に関わらず、**誰もが嫌がらせのない参加体験を得られるよう尽力します**。

---

## ✅ 望ましい行動

- **建設的なフィードバック**: 「動かない」だけでなく「再現手順 + 想定 + 実際」を伝える
- **5 大基本指針に沿った議論**: 機能要望・PR では各モード整合性 / 使えない機能改修 / a11y / 誰でも使いやすさ / チーム議論の観点で検討
- **業務担当者目線の尊重**: 開発者だけでなく業務担当者・管理者にとっての使いやすさを優先
- **謝意の表明**: ユーザー報告 / PR レビュー / ドキュメント改善に感謝を示す
- **異なる意見の尊重**: 技術的妥協点を探る (例: 「フル行番号ガター」より「カーソル位置インジケータ」など実用優先の判断)

---

## ❌ 受け入れられない行動

- 性的な言動・画像
- 侮辱的 / 中傷的なコメント / 個人攻撃 / 政治攻撃
- 公的または私的な嫌がらせ
- 明示的な許可なく他者の個人情報 (住所 / メール等) を公開
- **本拡張を悪用したアクション**: 他者の Salesforce 組織への無断アクセス、sid Cookie 流用、Login as User 機能の濫用
- その他、職場で不適切と判断される行為

---

## 🛡️ メンテナの責務

メンテナは行動規範違反に対し、適切な是正措置を取る権利と義務を持ちます:

- Issue / PR / コミットメッセージ / Wiki 等のコンテンツ削除・編集・拒否
- 不適切な行為を行った貢献者の一時的または永続的 ban
- 内容と理由を含めた決定の説明

---

## 📍 適用範囲

本行動規範は以下の場面で適用されます:

- このプロジェクトの GitHub Repository (Issue / PR / Discussions / Code)
- DevToolsNext を代表するすべての公開空間 (公式 SNS / 公式 Slack / イベント等)
- 個人がプロジェクト代表者として行動する場合

---

## 📞 報告

行動規範違反を見聞きした場合は、以下のいずれかで報告してください:

1. **推奨**: GitHub Security Advisory (Repository → Security → Advisories → "Report a vulnerability") — プライベートに扱えます
2. リポジトリオーナーへ直接連絡 (詳細は GitHub プロフィール参照)
3. SECURITY.md の脆弱性報告チャネルと同じフロー (24h 受領確認 / 7d 初期評価)

**報告者の個人情報は保護されます**。報告内容を公開する前に必ず報告者へ同意確認します。

---

## ⚖️ 是正措置のガイドライン

メンテナは以下の段階で対応します (Contributor Covenant v2.1 準拠):

| レベル | 違反内容 | 措置 |
|---|---|---|
| 1. 指摘 | 軽微な不適切言動 (1 回) | プライベート警告 + 公開謝罪の依頼 |
| 2. 警告 | 違反の継続 / 重大な単発違反 | 一定期間の交流制限 |
| 3. 一時 ban | 警告に従わない / 持続的攻撃 | 一定期間のプロジェクト参加禁止 |
| 4. 永久 ban | パターン化した違反 / 個人攻撃 / 個人情報暴露 | 永久的なプロジェクト参加禁止 |

---
