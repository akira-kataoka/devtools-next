            👥 ユーザー・ライセンス管理
            組織のユーザー数・ライセンス使用率・凍結ユーザー・MFA 設定状況・インストールパッケージを 1 画面で確認できます。本番組織でも参照のみのため安全です。
          

          
          
          
        

        


        

          
          

            

              
🏢 組織情報サマリ

              
              
            

            
エディション (Enterprise/Unlimited 等) / Sandbox 判定 / インスタンス / ストレージ使用量 / 通貨・タイムゾーン等の組織基本情報を 1 画面で確認。

            

              
取得ボタンをクリックすると、組織エディション / 環境種別 / ストレージ容量 / 通貨・言語・タイムゾーン を表示します。

            

          


          

            

              
📊 ユーザーライセンス使用状況

              
              
            

            
どのライセンスに何席残っているか即把握。年次更新前のライセンス調達判断、未使用ライセンスの棚卸しに使用。

            

              
取得ボタンをクリックすると、組織で有効な UserLicense ごとに 総数 / 使用中 / 残り / 使用率バー を表示します。

            

          


          

            

              
🔑 権限セットライセンス

              
              
            

            
Sales Cloud / Service Cloud などのアドオンライセンスの使用率と有効期限を表示。期限切れ前に再契約判断。

            

              
取得ボタンをクリックすると、PermissionSetLicense ごとの 総数 / 使用中 / 残り / 有効期限 / 使用率 を表示します。

            

          


          

            

              
👥 ユーザー集計

              
              
              
            

            
アクティブ / 非アクティブ / 凍結を即集計し、プロファイル別の人数分布も表示。「⏰ 未活動」ボタンで長期未ログインユーザーを抽出 (棚卸し・退職者特定用)。

            

              
取得ボタンをクリックすると、ユーザー総数 / アクティブ / 非アクティブ / 凍結数 およびプロファイル別人数を表示します。
「⏰ 30日/90日未活動」ボタンで長期未ログインユーザーを抽出できます。

            

          


          

            

              
❄️ 凍結ユーザー一覧

              
              
            

            
退職予定者の一時凍結・休職対応・不正アクセス疑い時の緊急ロック対象を一覧で確認。クリックで一括解除 Apex も呼出可。

            

              
取得ボタンをクリックすると、UserLogin.IsFrozen = true のユーザー一覧 (氏名 / ユーザ名 / プロファイル / IsActive) を表示します。

            

          


          

            

              
🛡️ MFA 設定状況

              
              
            

            
MFA を未設定のユーザー特定にも活用。Salesforce 強制要件 (内部ユーザの MFA) のコンプライアンス監査資料。

            

              
取得ボタンをクリックすると、MFA 設定済ユーザー数 と方式別 (TOTP / SalesforceAuthenticator / U2F) の内訳を表示します。
「🚨 未設定者を抽出」で、MFA を持たないアクティブ内部ユーザーを一覧化できます。

            

          


          

            

              
🔌 インストールパッケージ

              
              
            

            
AppExchange パッケージ・社内 ManagedPackage 一覧をバージョン付きで表示。バージョンアップ前の互換性把握に。

            

              
取得ボタンをクリックすると、InstalledSubscriberPackage (Tooling) からパッケージ名・名前空間・バージョンを一覧表示します。

  3. セキュリティ年次監査: 「🛡️ MFA 設定状況」+「🔐 ログイン履歴 (失敗のみ)」で未設定ユーザーと不正アクセス兆候を確認

    5.
