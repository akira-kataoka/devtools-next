🔗 API URL ビルダーの使い方 (3 ステップ)
  1. ① 操作を選ぶ: ドロップダウンから「何をしたいか」を選択 (例: get = 1 件取得 / query = SOQL 検索 / describe = 項目情報取得)
  2. ② 対象を入力: オブジェクト名 (例: Account) や ID (例: 0011x00000abcde) を入力
  3. ③ 「▶ URL 生成」をクリック: 右側に REST URL と curl コマンドが生成されます。「URL コピー」「curl コピー」でクリップボードにコピー、「ブラウザで開く」で直接アクセス可能
